2019 Summer Project Walden

在 Besttz 中提到的 Detox 逐渐成型。在友人的提醒下，朋友圈的预警多了一个新名“瓦尔登湖警告”, 那么这样的一场科技排毒就命名为 Project Walden 好了。在暑期的第一次尝试后，逐步可以推广至日常生活当中，每月都可以过上一次不受干扰的清净时光。
其实本项目中提到的排毒可以分为以下三种

  1. Social Detox “社交排毒” 旨于断开社交网络的影响，包含微信，Instagram 等等，这些正是过度推送打扰生活的主要来源。离开社交网络，并拒绝信息流渗入生活的同时，保持手机通话及短信的畅通。期间可以按照自己的喜好进行学习，健身，娱乐以及旅行。
  2. Internet Detox “互联网排毒” 旨于降低对互联网的依赖，更高强度的专注于生活。除必要情况外，完全切断互联网的链接，但依旧保持手机通讯，可以正常的和外界进行交流。每次进行排毒之前，对本次计划好的事项安排的必要资源进行离线储存，如数据文档，电子书，以及娱乐视频。
  3. Digital Detox “科技排毒” 离开电子产品的大胆尝试，其实认为应该不会执行这种对生活造成影响的计划。 在每次执行期间，由于没有社交网络的支持，可能需要以文字形式来规划并记录每日的生活。每一期的日记会以单独的页面展出，并在计划结束之后一并上传至本站点。项目执行的意义也在于降低自身对网络的需求，希望平日的生活也可以逐步改善，以更低的屏幕时间换取更高质量的生活。
